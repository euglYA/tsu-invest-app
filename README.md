# Веб-приложение "Платформа для прогнозирования капитализации инвестиционных фондов" по дисциплине "Введение в программную инженерию

Для установки нужных библиотек и модулей воспользуйтесь командой:
```bash
npm install
```

Для запуска приложения воспользуйтесь командой:
```bash
npm start
```

Приложение будет доступно по адресу http://localhost:3000 в браузере.
